## What to Build

The simulation session is where the expert role-plays every agent in the approved team
except the one being trained, while the learning agent practises coordination. Each turn:

1. The expert (playing `role`) sends a `message` to the learning agent (or vice versa,
   per the conversation flow -- in practice both directions occur).
2. The message is parsed into structured fields and validated against the receiving
   agent's contract via T1.10's `validate_message`.
3. If violations are found, the turn is recorded as `paused_for_violation=True` and the
   session cannot accept the next turn until each violation is resolved via
   `POST /simulations/{id}/violations/{index}/resolve`.
4. Observations (`learned`, `ambiguous`, `violation`) are appended to the session's
   running log.
5. On `POST /simulations/{id}/close`, all `coord_namespace_writes` accumulated during
   the session are flushed to Kuzu as `AgentRole` nodes and `HANDS_OFF_TO` edges in the
   coordination namespace.

Sessions are held in-memory (a process-local dict keyed by session id) for this task --
acceptable at apprentice-phase concurrency. If horizontal scaling of the API becomes
necessary, session state should move to Redis (same pattern as T0.3's operational state),
but that migration is out of scope here.

```python
# src/apprentice/simulation/session_manager.py

import json
import uuid
from datetime import datetime
from src.core.models import (
    SimulationSession, SimulationTurnInput, SimulationTurnResult,
    SimulationObservation, ContractViolation, AgentContract,
)
from src.core.enums import ObservationType
from src.apprentice.simulation.contract_validator import validate_message, apply_resolution
from src.apprentice.proposal_review_service import get_latest_proposal
from src.infra.kuzu_client import execute
from src.infra.postgres_client import get_pool

_sessions: dict[str, SimulationSession] = {}
_session_agents: dict[str, list[AgentContract]] = {}


class SessionNotFound(Exception):
    def __init__(self, session_id: str):
        self.session_id = session_id
        super().__init__(f"Simulation session '{session_id}' not found")


class SessionPausedForViolation(Exception):
    def __init__(self, session_id: str, unresolved_count: int):
        self.session_id = session_id
        self.unresolved_count = unresolved_count
        super().__init__(
            f"Session '{session_id}' has {unresolved_count} unresolved violation(s) -- "
            "resolve via /violations/INDEX/resolve before submitting another turn"
        )


async def start_session(agent_type: str) -> SimulationSession:
    """Start a new simulation session against the latest APPROVED composition."""
    proposal = await get_latest_proposal(agent_type)
    if proposal is None or proposal.status.value != "approved":
        raise ValueError(f"No approved composition for '{agent_type}' -- cannot start simulation")

    session = SimulationSession(
        agent_type_id=agent_type,
        composition_version=proposal.version,
    )
    _sessions[session.id] = session
    _session_agents[session.id] = list(proposal.agents)
    return session


def get_session(session_id: str) -> SimulationSession:
    session = _sessions.get(session_id)
    if session is None:
        raise SessionNotFound(session_id)
    return session


def submit_turn(session_id: str, turn_input: SimulationTurnInput,
                 to_agent: str, message_fields: dict) -> SimulationTurnResult:
    """
    Process one simulation turn.

    `turn_input.role` is who is SENDING this message.
    `to_agent` is the receiving agent's name -- validated against their contract.
    `message_fields` is the structured payload extracted from `turn_input.message`.
    """
    session = get_session(session_id)

    if session.status == "closed":
        raise ValueError(f"Session '{session_id}' is closed")

    unresolved = _count_unresolved_violations(session)
    if unresolved > 0:
        raise SessionPausedForViolation(session_id, unresolved)

    agents = _session_agents[session_id]
    violations = validate_message(message_fields, turn_input.role, to_agent, agents)

    turn_index = len(session.turn_history)
    observations: list[SimulationObservation] = []

    if violations:
        for v in violations:
            observations.append(SimulationObservation(
                type=ObservationType.VIOLATION,
                description=v.reason,
                turn_index=turn_index,
                roles_involved=[v.from_agent, v.to_agent],
                coord_edge_written=False,
            ))
    else:
        observations.append(SimulationObservation(
            type=ObservationType.LEARNED,
            description=f"{turn_input.role} -> {to_agent}: message conforms to contract",
            turn_index=turn_index,
            roles_involved=[turn_input.role, to_agent],
            coord_edge_written=False,
        ))

    result = SimulationTurnResult(
        turn_index=turn_index,
        role=turn_input.role,
        message=turn_input.message,
        violations=violations,
        observations=observations,
        paused_for_violation=len(violations) > 0,
    )

    session.turn_history.append(result)
    session.observations.extend(observations)
    session.current_role = turn_input.role

    return result


def resolve_violation(session_id: str, violation_index: int,
                       resolution: str, clarification: str | None) -> dict:
    """
    Resolve a pending violation from the most recent turn.

    `violation_index` indexes into the LAST turn's `violations` list (the only turn
    that can have unresolved violations, since submit_turn refuses new turns while
    any are unresolved).
    """
    session = get_session(session_id)
    if not session.turn_history:
        raise ValueError("No turns recorded yet")

    last_turn = session.turn_history[-1]
    if violation_index >= len(last_turn.violations):
        raise IndexError(f"violation_index {violation_index} out of range "
                          f"(turn has {len(last_turn.violations)} violations)")

    violation = last_turn.violations[violation_index]
    agents = _session_agents[session_id]

    updated_agents, coord_edge_written = apply_resolution(violation, resolution, clarification, agents)
    _session_agents[session_id] = updated_agents

    violation.resolution = resolution  # type: ignore[assignment]

    if coord_edge_written:
        session.coord_namespace_writes.append({
            "type": "input_schema_update",
            "agent_name": violation.to_agent,
            "field": violation.field,
            "field_type": clarification,
            "turn_index": last_turn.turn_index,
        })
        for obs in last_turn.observations:
            if (obs.type == ObservationType.VIOLATION and
                    violation.to_agent in obs.roles_involved and
                    violation.from_agent in obs.roles_involved):
                obs.coord_edge_written = True

    last_turn.paused_for_violation = _count_unresolved_violations(session) > 0

    return {"resolved": True, "coord_edge_written": coord_edge_written}


async def close_session(session_id: str) -> dict:
    """
    Close the session and flush coord_namespace_writes to Kuzu's coordination namespace.

    For each pending write:
      - Ensure an AgentRole node exists for the agent (MERGE-by-name pattern, as in T1.5)
      - For input_schema_update writes, update the AgentRole's input_schema JSON field
      - HANDS_OFF_TO edges between agents are created/updated based on observed
        successful handoffs (turns with no violations between two distinct roles)
    """
    session = get_session(session_id)
    if session.status == "closed":
        raise ValueError(f"Session '{session_id}' already closed")

    agent_type = session.agent_type_id
    agents = _session_agents[session_id]
    now_str = datetime.utcnow().isoformat()

    for agent in agents:
        _upsert_agent_role(agent_type, agent, now_str)

    coord_edges_written = 0
    for write in session.coord_namespace_writes:
        if write["type"] == "input_schema_update":
            coord_edges_written += 1

    handoffs_written = _write_handoff_edges(agent_type, session)
    coord_edges_written += handoffs_written

    violations_total = sum(len(t.violations) for t in session.turn_history)
    violations_unresolved = _count_unresolved_violations(session)

    session.status = "closed"
    session.closed_at = datetime.utcnow()

    return {
        "session_id": session_id,
        "coord_edges_written": coord_edges_written,
        "observations_captured": len(session.observations),
        "violations_total": violations_total,
        "violations_unresolved": violations_unresolved,
    }


def _count_unresolved_violations(session: SimulationSession) -> int:
    if not session.turn_history:
        return 0
    last_turn = session.turn_history[-1]
    return sum(1 for v in last_turn.violations if v.resolution == "pending")


def _upsert_agent_role(agent_type: str, agent: AgentContract, now_str: str) -> None:
    existing = execute(agent_type,
        "MATCH (r:AgentRole {name: $name}) RETURN r.id AS id",
        {"name": agent.agent_name}
    ).get_as_df()

    payload = {
        "name": agent.agent_name,
        "input_schema": json.dumps(agent.input_schema),
        "output_schema": json.dumps(agent.output_schema),
        "prohibited_fields": json.dumps(agent.prohibited_fields),
    }

    if len(existing) > 0:
        role_id = existing.iloc[0]["id"]
        execute(agent_type, """
            MATCH (r:AgentRole {id: $id})
            SET r.input_schema = $input_schema, r.output_schema = $output_schema,
                r.prohibited_fields = $prohibited_fields
        """, {"id": role_id, **payload})
    else:
        role_id = f"role-{uuid.uuid4().hex[:12]}"
        execute(agent_type, """
            CREATE (:AgentRole {
                id: $id, name: $name, input_schema: $input_schema,
                output_schema: $output_schema, prohibited_fields: $prohibited_fields
            })
        """, {"id": role_id, **payload})


def _write_handoff_edges(agent_type: str, session: SimulationSession) -> int:
    """
    For each pair of roles that exchanged at least one violation-free turn,
    write/update a HANDS_OFF_TO edge with confidence based on the proportion
    of clean turns between that pair.
    """
    pair_stats: dict[tuple[str, str], dict[str, int]] = {}

    for turn in session.turn_history:
        for obs in turn.observations:
            if len(obs.roles_involved) != 2:
                continue
            pair = tuple(obs.roles_involved)
            stats = pair_stats.setdefault(pair, {"clean": 0, "total": 0})
            stats["total"] += 1
            if obs.type == ObservationType.LEARNED:
                stats["clean"] += 1

    edges_written = 0
    for (from_role, to_role), stats in pair_stats.items():
        confidence = round(stats["clean"] / stats["total"], 3) if stats["total"] else 0.0
        if confidence <= 0:
            continue

        existing = execute(agent_type, """
            MATCH (a:AgentRole {name: $from_role})-[h:HANDS_OFF_TO]->(b:AgentRole {name: $to_role})
            RETURN h.confidence AS confidence
        """, {"from_role": from_role, "to_role": to_role}).get_as_df()

        if len(existing) > 0:
            execute(agent_type, """
                MATCH (a:AgentRole {name: $from_role})-[h:HANDS_OFF_TO]->(b:AgentRole {name: $to_role})
                SET h.confidence = $confidence
            """, {"from_role": from_role, "to_role": to_role, "confidence": confidence})
        else:
            execute(agent_type, """
                MATCH (a:AgentRole {name: $from_role}), (b:AgentRole {name: $to_role})
                CREATE (a)-[:HANDS_OFF_TO {
                    condition: $condition, validates: $validates, confidence: $confidence
                }]->(b)
            """, {
                "from_role": from_role, "to_role": to_role, "confidence": confidence,
                "condition": "observed during simulation",
                "validates": json.dumps([]),
            })
        edges_written += 1

    return edges_written
```

### API Routes

```python
# src/api/routes/simulations.py

from fastapi import APIRouter, HTTPException
from src.apprentice.simulation.session_manager import (
    start_session, get_session, submit_turn, resolve_violation, close_session,
    SessionNotFound, SessionPausedForViolation,
)
from src.core.models import SimulationTurnInput

router = APIRouter(prefix="/simulations", tags=["simulations"])


@router.post("", status_code=201)
async def create_session(body: dict):
    try:
        return await start_session(body["agent_type_id"])
    except ValueError as e:
        raise HTTPException(status_code=409, detail=str(e))


@router.get("/{session_id}")
async def read_session(session_id: str):
    try:
        return get_session(session_id)
    except SessionNotFound as e:
        raise HTTPException(status_code=404, detail=str(e))


@router.post("/{session_id}/turn")
async def post_turn(session_id: str, body: dict):
    turn_input = SimulationTurnInput(role=body["role"], message=body["message"])
    to_agent = body["to_agent"]
    message_fields = body["message_fields"]

    try:
        return submit_turn(session_id, turn_input, to_agent, message_fields)
    except SessionNotFound as e:
        raise HTTPException(status_code=404, detail=str(e))
    except SessionPausedForViolation as e:
        raise HTTPException(status_code=409, detail=str(e))


@router.post("/{session_id}/violations/{violation_index}/resolve")
async def resolve_turn_violation(session_id: str, violation_index: int, body: dict):
    try:
        return resolve_violation(
            session_id, violation_index,
            resolution=body["resolution"], clarification=body.get("clarification"),
        )
    except SessionNotFound as e:
        raise HTTPException(status_code=404, detail=str(e))
    except (ValueError, IndexError) as e:
        raise HTTPException(status_code=400, detail=str(e))


@router.post("/{session_id}/close")
async def close(session_id: str):
    try:
        return await close_session(session_id)
    except SessionNotFound as e:
        raise HTTPException(status_code=404, detail=str(e))
    except ValueError as e:
        raise HTTPException(status_code=409, detail=str(e))
```

## Acceptance Criteria

1. `start_session()` against an agent type with no `approved` composition raises `ValueError`
2. A turn with a compliant message produces one `LEARNED` observation and `paused_for_violation=False`
3. A turn with violating fields produces `VIOLATION` observations per field, `paused_for_violation=True`, and `resolution="pending"` on each `ContractViolation`
4. Calling `submit_turn` again while violations are unresolved raises `SessionPausedForViolation`
5. `resolve_violation(..., resolution="clarify_contract", clarification="int")` updates the in-session agent contract, sets `coord_edge_written=True` on the corresponding observation, and unblocks `submit_turn`
6. `resolve_violation(..., resolution="mark_edge_case", ...)` unblocks `submit_turn` without modifying any contract
7. `close_session()`:
   - Writes one `AgentRole` node per agent in the (possibly-edited) composition
   - Writes `HANDS_OFF_TO` edges for every pair of roles that had at least one clean turn, with `confidence = clean_turns / total_turns` for that pair
   - Returns `violations_unresolved == 0` when all violations were resolved before close
8. Turn round-trip (`submit_turn` call to result) completes in under 2 seconds when `validate_message` runs against an in-memory `agents` list (no I/O in the hot path)

## Notes

- **Session state is in-memory and process-local.** This is acceptable because
  simulation sessions are inherently interactive and synchronous -- an expert is
  actively driving the conversation in real time. If the API ever runs with multiple
  worker processes behind a load balancer, sessions need sticky routing or migration to
  Redis (the key pattern `session:{session_id}:*` would follow T0.8's conventions).
  This task card does not implement that migration.
- **`_session_agents` holds the mutable, possibly-edited copy of the approved
  composition's agent contracts for the duration of the session.** The original
  `team_compositions` row (Invariant I6, immutable after signature) is never touched
  during simulation -- contract clarifications discovered during simulation are staged
  in-memory and only become permanent via the Kuzu `AgentRole.input_schema` write at
  `close_session()`. If a structural team composition change is warranted (not just a
  contract clarification), that goes through T1.8's edit flow as a new version, which
  is a separate, expert-initiated action outside this session.
- **`HANDS_OFF_TO` confidence is `clean_turns / total_turns` for each role pair,
  computed at close time, not incrementally.** This is intentionally simple for a single
  session. If an agent type runs multiple simulation sessions over its apprenticeship,
  `_write_handoff_edges`'s `SET h.confidence = ...` will overwrite the prior session's
  confidence rather than averaging across sessions. This mirrors T1.5's "confidence
  reflects latest evidence" pattern for the task namespace, applied to the coordination
  namespace -- the most recent simulation is the most relevant evidence of current
  contract behavior.
- **Why `coord_namespace_writes` is tracked separately from the Kuzu write itself:**
  it gives `close_session()`'s return summary (`coord_edges_written`) a clean count
  independent of how many `AgentRole`/`HANDS_OFF_TO` writes actually executed, and gives
  the simulation console (the three-panel UI from earlier design discussion) a list of
  "what changed" to display to the expert before they confirm closing the session.
- **2-second round-trip target excludes Kuzu writes** -- `submit_turn` only validates
  against the in-memory `agents` list (T1.10, pure Python). All Kuzu writes happen at
  `close_session()`, which has no latency requirement since it's a one-time end-of-session
  operation, not a per-turn one.
