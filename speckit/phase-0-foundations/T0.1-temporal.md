## What to Build

This task validates that the Temporal cluster is reachable and a Python worker can register
and execute a workflow end-to-end. No agent logic yet — just prove the plumbing works.

### Minimal Hello-World Workflow

```python
# src/temporal/workflows.py (initial version — extended in T2.1/T2.2)

from datetime import timedelta
from temporalio import workflow, activity


@activity.defn
async def hello_activity(name: str) -> str:
    return f"Hello, {name}!"


@workflow.defn
class HelloWorkflow:
    @workflow.run
    async def run(self, name: str) -> str:
        return await workflow.execute_activity(
            hello_activity,
            name,
            start_to_close_timeout=timedelta(seconds=10),
        )
```

### Worker Entrypoint

```python
# src/temporal/worker.py

import asyncio
import os
from temporalio.client import Client
from temporalio.worker import Worker
from src.temporal.workflows import HelloWorkflow, hello_activity

TEMPORAL_HOST = os.getenv("TEMPORAL_HOST", "localhost:7233")
TEMPORAL_NAMESPACE = os.getenv("TEMPORAL_NAMESPACE", "agent-platform")


async def main():
    client = await Client.connect(TEMPORAL_HOST, namespace=TEMPORAL_NAMESPACE)

    worker = Worker(
        client,
        task_queue="agent-platform-tasks",
        workflows=[HelloWorkflow],
        activities=[hello_activity],
    )

    print(f"Worker started. Connected to {TEMPORAL_HOST}, namespace={TEMPORAL_NAMESPACE}")
    await worker.run()


if __name__ == "__main__":
    asyncio.run(main())
```

### Namespace Registration

Run once after the Temporal container is healthy:

```bash
docker exec alf-temporal \
  temporal operator namespace create agent-platform --retention 30d
```

Verify:

```bash
docker exec alf-temporal temporal operator namespace describe agent-platform
```

## Acceptance Criteria

1. `docker compose up -d temporal temporal-ui` brings up both containers healthy within 30 seconds
2. `temporal operator namespace create agent-platform --retention 30d` succeeds (or returns "already exists" on re-run)
3. Temporal UI at `http://localhost:8088` shows the `agent-platform` namespace
4. Starting `src/temporal/worker.py` registers successfully (no connection errors)
5. A client can start `HelloWorkflow` and receive `"Hello, World!"` as the result within 5 seconds
6. Killing the worker process mid-execution and restarting it does not lose the workflow (Temporal durability check)

## Test

```python
# tests/phase_0/test_infra_connectivity.py (additions)

import pytest
import asyncio
from temporalio.client import Client
from temporalio.worker import Worker
from src.temporal.workflows import HelloWorkflow, hello_activity

TEMPORAL_HOST = "localhost:7233"
NAMESPACE = "agent-platform"


@pytest.mark.asyncio
async def test_temporal_namespace_reachable():
    """Confirm we can connect to the Temporal cluster and namespace exists."""
    client = await Client.connect(TEMPORAL_HOST, namespace=NAMESPACE)
    assert client is not None


@pytest.mark.asyncio
async def test_hello_workflow_executes():
    """End-to-end: start a worker, run a workflow, get the result."""
    client = await Client.connect(TEMPORAL_HOST, namespace=NAMESPACE)

    async with Worker(
        client,
        task_queue="test-hello-queue",
        workflows=[HelloWorkflow],
        activities=[hello_activity],
    ):
        result = await client.execute_workflow(
            HelloWorkflow.run,
            "World",
            id="test-hello-workflow-1",
            task_queue="test-hello-queue",
        )

    assert result == "Hello, World!"


@pytest.mark.asyncio
async def test_workflow_history_visible_in_ui():
    """Workflow history should be queryable via the client (proxy for UI visibility)."""
    client = await Client.connect(TEMPORAL_HOST, namespace=NAMESPACE)
    handle = client.get_workflow_handle("test-hello-workflow-1")
    desc = await handle.describe()
    assert desc.status.name in ("COMPLETED", "RUNNING")
```

## Notes

- The `HelloWorkflow` is throwaway scaffolding for this task. T2.2 replaces it with `AgentTaskWorkflow`
  and T1.x apprentice sessions add `ApprenticeSessionWorkflow`. Do not delete `workflows.py` —
  extend it.
- Task queue name `agent-platform-tasks` is the default for all production workflows. Tests use
  isolated queue names (e.g. `test-hello-queue`) to avoid colliding with the long-running worker.
- If `temporal operator namespace create` returns `AlreadyExists`, that's success — the command
  is idempotent in practice (just check the error message, don't fail the setup script on it).
